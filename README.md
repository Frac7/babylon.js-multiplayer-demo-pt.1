# babylon.js-multiplayer-demo-pt.1

1. `cd server`, `yarn install`, `yarn start`;
2. `cd client`, `yarn install`, `yarn start`.

Open `localhost:3000`; to connect with another player, open a new tab and connect to `localhost:3000`.
